wave svg
----------------


#### API

```js
  new waveSvg({
    buffer: Audiobuffer,
    maxHeight: 300,
    width: 1300, <default window.innerWidth>
    pixelsPerSecond: 200, <default window.innerWidth>
    appendTo: dom node, <default document.body>
    max: 0.2 <defaults to the tallest audio peak>
  });
```

#### To Dev
```
  cake watch
```

#### To Build From Source
```
  cake build
```
